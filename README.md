# Survival-AI
Survival AI es una aplicación de supervivencia con guías prácticas, simulador de decisiones, IA integrada, mochila interactiva, desafíos con fotos, sistema de niveles, XP, logros, estadísticas y expediciones. Diseñada para aprender, practicar y mejorar habilidades de supervivencia de forma interactiva y sin conexión.
